# College Facts Agent

A natural-language agent over a curated catalog of U.S. college and
university facts: IPEDS + Academic Insights + Common Data Set + College
Scorecard + EADA + multiple rankings (US News, Washington Monthly, Forbes).

## Quick start

```bash
# 1. R ETL: data/ -> output/*.csv
Rscript etl/run_all.R

# 2. Loader: output/*.csv -> Postgres
cp .env.example .env       # fill in DB_DSN, API keys, etc.
python load/load_to_postgres.py

# 3. Agent: serve /chat
cd agent
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
uvicorn app:app --reload --port 8000
```

## Layout

- `etl/` — R pipelines (one per facts theme) + Python scrapers for the
  non-IPEDS sources.
- `data/` — raw inputs; self-contained so the repo can be moved to a server
  without external dependencies (≈700MB).
- `output/` — built CSVs; gitignored, regenerated by `etl/run_all.R`.
- `load/` — Postgres schema + Python loader.
- `agent/` — FastAPI service, queries the Postgres catalog.
- `docs/` — provenance, year conventions, codebook ingestion.
